Rental logistics — model & standard features

Supports a job with multiple locations, each holding multiple units; tracks self-driven mileage (towed units excluded); and adds the rental-industry standards that turn this into a real fleet-hire operation.

New entities: HireLocation (belongs to a project hire; name, place/geo, dates, crew count, status) and HireAllocation (asset ↔ location, with check-out/return readings). An asset can move between locations via a transfer leg, which is what animates on the map.

2 · Mileage — self-driven only

Per-vehicle odometer
Check-out & return readings on motorized assets → miles driven this hire, plus per-leg deltas. Trailers show "moved by <tow vehicle>".
Fuel efficiency
km ÷ litres from fuel logs → km/L per vehicle; flags sudden drops (mechanical issue or fuel leakage/theft).
Mileage allowance & excess
Optional included-km per hire; auto-flag & chargeable excess (e.g. Sprinter ⚠ above).
Service triggers
Odometer feeds the existing Preventive Maintenance schedule — "service due in 800 km".

3 · Rental-industry standards worth adding

core Check-out / Check-in inspection
Walkaround with photos at dispatch & return; sets the damage baseline, auto-creates an Incident on new damage.
core Hire lifecycle
Reserved → Dispatched → On-hire → Off-hire → Returned, with timestamps; early-return proration.
core Rate cards
Day / week / month rates, minimum hire, delivery & collection fees — feeds the quote & the production budget.
core Utilization & idle
% on-hire vs idle per asset; "idle on location 3 days" nudge to off-hire and save cost.
field Geofence per location
Alert if a unit leaves its assigned location/route; the off-route alert on the map.
field Driver compliance gate
Block dispatch if driver licence/visa expired (ties to Compliance & Renewals).
field Fuel policy
Full-to-full; refuel charge if returned low; fuel-card reconciliation against odometer.
nice Damage waiver / insurance
Per-hire waiver option & excess; attached to the contract.
nice Consumables on hire
Jerry cans, cables, leveling blocks issued with a unit; reconciled on return (ties to Inventory).
nice Cost allocation
Roll hire cost up per location → per project → into the production budget cost report.
